## CONTENT SPECIFICATIONS

### Format Standards
- **File Format:** Markdown (.md)
- **Naming:** _index.md for section pages
- **Frontmatter Required:**
  ```yaml
  ---
  title: "Topic Title"
  description: "SEO-optimized description (150-160 chars)"
  date: "2026-01-04"
  weight: 10
  tags: ["tag1", "tag2", "tag3"]
  categories: ["category"]
  ---
  ```

### Content Requirements
- **Length:** 500-900 words per page
- **Structure:**
  - H1: Page title (from frontmatter)
  - H2: Main sections
  - H3: Subsections
- **Elements:**
  - Bullet points for clarity
  - Tables for comparisons/data
  - Mathematical formulas (LaTeX format: `$formula$` inline, `$$formula$$` block)
  - Practical examples
  - Design considerations
  - Best practices

### SEO Optimization
- **Title:** 50-60 characters, includes primary keyword
- **Description:** 150-160 characters, includes 1-2 keywords
- **Headers:** Include keywords naturally
- **Internal Linking:** Link to related topics
- **Image Alt Text:** Descriptive, includes keywords
- **URL Structure:** Hierarchical, keyword-rich

### Anti-Plagiarism Strategy
- Original physics-based explanations
- Unique calculation examples
- Custom diagrams and illustrations
- Proprietary methodology presentations
- Cite ASHRAE standards as reference frameworks
- Present information through engineering analysis lens

### Formulas and Diagrams
- **Formulas:** LaTeX syntax for mathematical expressions
- **Diagrams:** Mermaid.js for flowcharts and diagrams
- **Tables:** Markdown tables for data presentation
- **Images:** SVG preferred, descriptive filenames

### Technical Accuracy
- Based on ASHRAE Handbooks (2021-2024)
- Engineering fundamentals from first principles
- Industry standard calculation methods
- Code compliance references (IMC, IBC, ASHRAE standards)
- Metric (SI) and Imperial (IP) units
